Welcome to Worldcraft++

This is the central hub for the upcoming custom build of the leaked Hammer Editor. Modeled after the philosophy of modern community tools like Hammer++, this project aims to specifically stabilize and tailor the editor for the legendary 2003 leaked build of Half-Life 2 (WC Mappack / Anon Leak).

Stay tuned for updates, documentation, and the eventual public beta release. Feel free to navigate using the tabs above to track our progress.

Worldcraft++ is a custom build of Leaked Hammer Editor created by FILipKOS. • Not affiliated with Valve Software or Hammer++!
