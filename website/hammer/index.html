Welcome to the Worldcraft++

This is the central hub for the upcoming custom build of the leaked Hammer Editor. Modeled after the modern community tools like Hammer++ and J.A.C.K, this project aims to specifically stabilize and tailor the editor for the 2003 leaked build of Half-Life 2.

Stay tuned for updates, documentation, and the eventual public beta release! Feel free to navigate using the tabs above to track our progress.
